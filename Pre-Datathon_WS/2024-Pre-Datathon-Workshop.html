Better Data Visualizations

Five guidelines for better data visualizations from Jon Schwabishâs book called Better Data Visualizations.

  • Show the data
  • Reduce the clutter
  • Integrate the graphics and text
  • Avoid the spaghetti chart
  • Start with gray

Guideline 2: Reduce the Clutter

âThe use of unnecessary visual elements distracts your reader from the central idea and clutters the page.â

A few things to avoid:

  • heavy tick marks and grid lines
  • unnecessary 3d
  • excessive text

Consider this image from Claus Wilkeâs Fundamentalâs of Data Visualization.

Guideline 3: Integrate the Graphics and Text

  1. Remove legends when possible and label data directly
  2. Write an active title like a newspaper headline
  3. Add explainers

labs() adds title, subtitle, caption, and tag to ggplot2 objects. It can also be used to overwrite x, y, and legend titles. Use NULL to remove a label entirely (not "").

Guideline 4: Avoid the Spaghetti Chart

âSometimes we face the challenge of including lots of data in a single graph but we donât need to try to pack everything into a single graph.

Faceting or using small multiples is a useful way to de-clutter a busy data visualization. Weâve already encountered faceting multiple times because it is so natural in ggplot2. With effective small multiples, if a reader understand how to read one small multiple then they should understand how to read all of the multiples. Two tips:

  1. Arrange the small multiples in a logical order
  2. Use the same layout, size, font, and color in each small multiple

Steps for Graphing Calculating Cumulative Crime Incidence

Step 1: Compute the cumulative sum of each crime incidence type over the years.

UC.cumsum <- UC %>% 
  count(CRIME.CATEGORY.DESCRIPTION, YEAR.OCCURRED) %>%
  group_by(CRIME.CATEGORY.DESCRIPTION) %>% 
  mutate(cumsum = cumsum(n))

Step 2: Create a line plot with x = YEAR.OCCURRED, y = cumsum, group = CRIME.CATEGORY.DESCRIPTION, and geom_line()

UC.cumsum %>% 
  ggplot(aes(YEAR.OCCURRED, cumsum, group =CRIME.CATEGORY.DESCRIPTION))+
  geom_line()

Note: If we simply create a line graph with color = CRIME.CATEGORY.DESCRIPTION, then
